# svelte-spa
Demo Application using SvelteJS

## Clone and Install dependencies

```bash
git clone https://github.com/keuller/svelte-spa

cd svelte-spa

npm install
```

## Running Dev
```bash
$ npm start
```

## Building to Production
```bash
$ npm run build
```

## Running Production Bundle
```bash
$ npm run serve
```

## Running Tests
```bash
$ npm run test
```
